现在是2021年4月7日，yan突然发现自己并不会写稿。我想了好一会儿才写出这第一句话。删掉重写。

高三的我曾试着把脑子里的黄色废料梳理成一篇可以投稿在某色情「文学」网站的文章。因为画的饼太大，我失去了完成它的动力，它成了 未完成品。我创作过好多好多 未完成品 ，它们现在要么是躺在硬盘上，要么是堆在活页册里。我把之前用 hexo 搭建的那个仅仅用来分享了《C Primer Plus》学习笔记的博客删掉，是因为我知道自己已经完成不了它了。它也是 未完成品 。我没能进学校的 Linux User Group。这段经历也是 未完成品 。Notion 某个 Page 里罗列着的视频计划，高考前为即将到来的暑假画的「分镜」们，都成了 未完成品 。我打算重修一遍大一，我也是 未完成品。

现在是2021年4月7日，yan突然发现自己并不会喝可乐。在用左手打这段话的同时，右手上的可乐撒了，有几滴溅在了白衣服上。我喜欢喝零度，但今天买的是百事，这可能是潜意识在劝我别干蠢事吧。实在是不应该呢。

现在是2021年4月7日，yan突然发现自己并不会学习。我极度缺乏自制力。对了，我刚才是在看网课来着。想起 @vonhyou 同学推荐的 Learning How to Learn，我需要去学习一下。希望我不需要去学习Learning How to Learn How to Learn。

说说yan完成了哪些事吧。感谢 @vonhyou 同学推荐，我找到了 Coursera 这个网站，并在其上成功完成了一门课程（HTML5 入门）的学习。我很高兴地发现我之前打算看 Document 学习一项编程技能的想法还是 Too Young Too Simple 了，起码对我这样不会学习的人来说还是 Naïve 了。我之前写的 HTML 竟然连 DOCTYPE 都没有！感谢浏览器不杀之恩。

我在 The Type 的 这个页面 找到了 西安 Localand 书店，相见恨晚。我成功在它楼上的一家中餐馆用一杯美式和一台电脑坐了一下午，出门回头看牌匾才知道那竟然是家中餐馆，在门外面直行才是咖啡店。在原岩咖啡买了500G中深烘的拼配，回宿舍泡杯咖啡，打翻了法压壶，清理半个多小时，并成功烫伤大腿。

于是我买了一套手冲咖啡的器具。还没到。这次特意买的塑料的 Hario V60，希望它的服役时间能长一点。才不是因为玻璃的 V60 贵一百多块钱。

写这篇博客的时候我还不会写CSS，所以你看到的这个页面一定不会好看。其实这句话有点矛盾：因为本网站流量基本为 0，并不会有人看到这个页面最初的样子。所以我会把它的最初版本 放在这里，希望你点开的时候发现我进步了很多。希望它不会成为未完成品。
